# Rust 程序设计语言

_作者：Steve Klabnik、Carol Nichols、Chris Krycho，以及 Rust 社区贡献者_

本版本内容假设你使用 Rust 1.90.0（发布于 2025-09-18）或更高版本，并在所有
项目的 *Cargo.toml* 中使用 `edition = "2024"`，以启用 Rust 2024 Edition 的惯用写法。
有关安装或升级 Rust 的说明，请参见[第 1 章“安装”][install]<!-- ignore -->；
有关版本（Edition）的说明，请参见[附录 E][appendix-e]<!-- ignore -->。

HTML 格式可在线阅读：
[https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/)
也可通过 `rustup` 安装的离线文档阅读；运行 `rustup doc --book` 即可打开。

社区还提供了多个[翻译版本][translations]。

本书还提供 [No Starch Press 的纸质版与电子版][nsprust]。
